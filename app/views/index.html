Roadmap

Goal: Plan and track future development of the Digital Identity service.

! Warning This roadmap is indicative and subject to change based on user needs, policy requirements, and technical considerations.

Current Quarter (Q4 2025)

  • In progress Enhanced biometric authentication

    Implement facial recognition and fingerprint support for mobile devices to improve security and user experience.

  • Planned API rate limiting improvements

    Upgrade infrastructure to handle 10M+ monthly authentications with improved rate limiting and caching.

  • Planned Accessibility audit and remediation

    Conduct comprehensive WCAG 2.2 AA audit and implement recommendations.

Q1 2026 (Jan - Mar)

  • Not started Document verification service integration

    Connect with Passport Office and DVLA systems for automated document verification.

  • Not started Multi-language support

    Add Welsh, followed by translations for most common languages used by UK residents.

  • Not started Advanced fraud detection

    Deploy machine learning models for real-time fraud pattern detection and prevention.

Q2 2026 (Apr - Jun)

  • Not started Local authority integration pilot

    Pilot program with 5 local authorities for council services integration.

  • Not started Self-service identity management

    Enable users to update personal details, manage security settings, and view authentication history.

  • Not started Developer portal launch

    Public API documentation, sandbox environment, and onboarding tools for government departments.
